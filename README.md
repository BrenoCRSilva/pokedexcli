# 🎮 Pokédex CLI — *O Jogo no Terminal!*

Bem-vindo à **Pokédex CLI**, um jogo interativo feito em Go que transforma o seu terminal em uma verdadeira aventura Pokémon!  
Explore mapas, descubra Pokémons selvagens, capture-os e monte sua própria coleção — tudo isso com comandos simples e nostálgicos.

---

## 🧩 O que é isso?

A Pokédex CLI é mais do que uma simples consulta de informações.  
Aqui, **você joga**:

- 🌍 Navegue entre mapas conectados
- 🧭 Descubra Pokémons em rotas diferentes
- 🥇 Capture Pokémons com o comando `catch`
- 📖 Veja sua coleção com `pokedex`
- ❓ Explore todos os comandos com `help`
- ⚙️ Tudo direto do terminal, com dados em tempo real da [PokéAPI](https://pokeapi.co/)

---

## 📦 Como jogar

### 1. Clone o repositório:

```bash
git clone https://github.com/BrenoCRSilva/pokedexcli.git
cd pokedexcli
```

### 2. Compile o projeto:

```bash
go build
```

### 3. Execute:

```bash
./pokedexcli
```

---

## 💬 Comandos disponíveis

Digite `help` a qualquer momento para ver a lista completa de comandos!

Alguns exemplos:

```bash
help            # Mostra todos os comandos disponíveis
map             # Mostra a localização atual e caminhos disponíveis
explore         # Explora o mapa atual e revela Pokémons selvagens
catch <nome>    # Tenta capturar um Pokémon
pokedex         # Lista os Pokémons capturados
```
